# pas-docker-mysql
Run docker-mysql with pas

Run :
```
git clone git@github.com:wahyutaufik/pas-docker-mysql.git
cd pas-docker-mysql
pas up
```

Open http://localhost:49161/phpmyadmin in your browser with following credential:
```
username: root
password: password
```

Login by SSH
```
ssh root@localhost -p 49160
password: admin
```
